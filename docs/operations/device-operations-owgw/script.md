Use Script to run shell commands or bundled command sets on the device.

## How to Use

- Open the Actions menu for the device.
- Select Script.
- Choose the type: Shell for single commands or Bundle for multiple commands.
- Optionally enable Wait for result.
- Enter or upload the script, then click Start.

![Script execution setup](/img/operations/mango-cloud-operations-guide/image45.webp)

![Script execution results](/img/operations/mango-cloud-operations-guide/image58.webp)

## Result

- The device executes the provided script or bundle.
- After the command completes successfully, you can review, copy, or download the response.
