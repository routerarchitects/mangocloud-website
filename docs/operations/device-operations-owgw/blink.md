- Select Blink from the actions menu.

![Blink action - device selection](/img/operations/mango-cloud-operations-guide/image6.webp)

- A prompt confirms success and the device blinks for a fixed duration.

![Blink action - confirmation](/img/operations/mango-cloud-operations-guide/image3.webp)

Verify:

- Check the Commands section for a timestamp matching your action.
- Click View Details and check for "result":"done".

![Blink action - command result](/img/operations/mango-cloud-operations-guide/image44.webp)
