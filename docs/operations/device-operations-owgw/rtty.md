Use RTTY to open a remote terminal session to the device.

## How to Use

- Click the **Connect** button to start an RTTY session.
- Log in with user `root` and password `openwifi`.

![Rtty connection](/img/operations/mango-cloud-operations-guide/image42.webp)

## Result

- A remote terminal session is established with the device.
- You can use the terminal to run commands for troubleshooting or administration.
