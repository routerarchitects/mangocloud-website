---

Use Telemetry to collect real-time operational data from the device.

## How to Use

- Open the Actions menu for the device.
- Select Telemetry.
- Set the interval and duration.
- Click Start to collect real-time data.

![Telemetry command output](/img/operations/mango-cloud-operations-guide/image41.webp)

## Result

- The command returns live telemetry data such as CPU, memory, clients, and Wi-Fi status.
- After the collection completes, you can review and copy the output data.
