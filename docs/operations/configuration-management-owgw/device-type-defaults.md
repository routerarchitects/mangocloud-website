title: Device-type defaults
---

Steps:

1. Log in to OWGW-UI.
2. Go to Defaults -> Configurations.

![Defaults - navigation](/img/operations/mango-cloud-operations-guide/image7.webp)

3. Click Create and add a default configuration:
   - Name (e.g., `tplink_default_configuration`)
   - Device type
   - Configuration JSON

![Defaults - create configuration](/img/operations/mango-cloud-operations-guide/image13.webp)

4. Click Save.
5. Confirm the new entry appears in the Defaults list.

![Defaults - configuration list](/img/operations/mango-cloud-operations-guide/image53.webp)
