title: Common Design Patterns
---

Common configuration design patterns help you place settings at the right level for reuse and maintainability.

## Entity Level

Define common settings shared across all devices, for example:

- Default `WAN`
- `SSIDs`
- `Services`

## Venue Level

Apply site-specific configurations, for example:

- Location-based `SSIDs`
- Network settings

## Device Level

- Use for one-off overrides on individual devices
