To deploy Mango Cloud, go through the official deployment repository and follow the README instructions to complete the setup.

## Deployment Guide

- Visit the deployment repository:  
  https://github.com/routerarchitects/mango-cloud-deployment/tree/release/v1.0.0

- Follow the README step-by-step to set up the Mango Cloud stack using Docker Compose

- Update certificates as required for secure communication

- Start the stack and verify that all services are up and accessible

## Key Notes

- The repository includes all core services such as controller, provisioning, analytics, and UI components
- Ensure Docker and Docker Compose are installed before starting
- For production setups, configure domain and certificates appropriately
