This release includes improvements across configuration, device management, monitoring, and user experience to enhance usability and system reliability.

## Gateway Configuration UI Enhancement
- Gateway configuration can now be edited using a card-based UI with sectioned forms
- Replaces raw JSON editing, improving usability and reducing configuration errors

## Configuration Management Enhancement
- Introduced a configuration table for managing settings
- Supports device-group–specific configurations (OLG / AP / Switch)
- Improves organization and scalability

## Device Management Workflow Improvements
- Clear separation between gateway and mesh devices
- Ensures structured and predictable device management
- Mesh devices must be removed before deleting the gateway device

## Monitoring Workflow Improvement
- Monitoring is automatically enabled when a gateway device is added
- Monitoring is automatically stopped when no devices are present

## Multi-Interface Management
- Subscriber and device management supported via:
  - Mobile App
  - OWPROV UI
- Provides better flexibility and operational accessibility
