# Contributing

MangoCloud welcomes contributions from Router Architects partners and the broader OpenWiFi community.

## Process

1. Review open roadmap items and discuss ideas in GitHub Discussions.
2. Fork the repository and create feature branches per contribution.
3. Include tests and documentation updates in every PR.
4. Request reviews from maintainers and respond to feedback promptly.

## Guidelines

- Follow the coding standards documented in the repository.
- Keep changes scoped and reference related issues.
- Ensure all commits have a Signed-off-by line complying with the Developer Certificate of Origin.

## Communication

- Discussions: https://github.com/RouterArchitects/mangocloud-community/discussions
- Security: security@routerarchitects.com
- Release Announcements: posted via Discussions and mailing list.
